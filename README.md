# DosWin32
Win32 operating system for MS-DOS  

Core System Components
* KERNEL32.DLL
  * Threading
  * File management
  * Memory management
    * Heap (global & local)
    * Virtual Memory
      * Memory-mapped files
* GDI32.DLL
* USER32.DLL
* Display drivers
  * DIB display driver
  * VGA display driver
